# mrsisa_vojvodic

Author: Smiljana Vojvodic SW6/2017

Technologies used: Java Spring, JQuery, PostgreSql

Running the application:
1. Import project to Eclipse: Import -> Maven -> Existing Maven Project -> project name
2. Install all dependencies from pom.xml: right click on project -> Run As -> 4 Maven build... -> into "Goals" type "clean compile install" -> Run
3. Running the program: right click on project -> Run As -> Java Application
4. Type into browser: http://localhost:8081/
